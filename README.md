### About

这个项目是我在 WangXuTech 期间为 GitMind 编写的脑图核心库.

GitMind 项目在早期使用 kityminder 作为核心库, 但是其代码混乱, BUG 众多, 而且无法扩展. 所以我编写了这个脑图核心库, 于 2021 年更换.

这个脑图核心库已经完全脱离了 kityminder, 所有的东西都被重新设计, 仅有数据格式和概念以及术语较为接近.
